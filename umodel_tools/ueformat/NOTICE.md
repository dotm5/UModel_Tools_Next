UEFormat Attribution
====================

The binary layout and parser structure in this package are based on the
UEFormat Blender Plugin distributed in `reference/UEFormat-blender.zip`.

Upstream add-on metadata:

- Name: UE Format (.uemodel / .ueanim)
- Maintainer: https://github.com/h4lfheart
- License: SPDX:GPL-3.0-or-later

This package currently implements only the `.uemodel` reader/import path used
by UModel Tools v1. `.ueanim` and `.uepose` are intentionally out of scope.
